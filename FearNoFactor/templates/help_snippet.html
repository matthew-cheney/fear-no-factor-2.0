Instructions

This activity is designed to strengthen your integer factoring skills ("reverse multiplication"). Each exercise presents you with a random number. Your task is to enter all unique pairs of numbers which multiply to produce it.

For example, the number 24 can be produced by four pairs:

1 x 24

2 x 12

3 x 8

4 x 6

In landscape mode, enter the first number in each pair using the left keypad and the other number using the right keypad. Then select either green enter ("E") key. If the pair is valid, it will be displayed in the middle.

In portrait mode, select the left blue box (with rounded corners) and key in one number then select the right blue box and key in the other number. Then select the enter (green "E") key. If the pair is valid, it will be displayed in the upper right.

Select the Submit button when you have identified all pairs.

Select the "Basic" button to increase the difficulty level. Select "Advanced" to return to the basic level.

You are required to log in using your Google account so that your record of exercises completed can be preserved between sessions on any supported device.
